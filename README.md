# Smart Food Redistribution Platform

### Connecting Donors, NGOs & Volunteers in Real-Time

---

## CodeSangram – Online Hackathon 2026

Developed as part of **CodeSangram Online Hackathon 2026**, organized by **Poornima University (PU), Jaipur**.

The **Smart Food Redistribution Platform** is a web-based system designed to minimize food wastage by enabling real-time coordination between food donors, NGOs, and volunteers.

---

## Project Vision

Every day, large quantities of food are wasted while many people struggle with hunger.
This platform bridges the gap by creating a **smart digital connection** between surplus food sources and communities in need.

---

## Problem Statement

* Excess food from events and households goes to waste.
* NGOs struggle to find available food quickly.
* Volunteers lack coordination for food collection and delivery.
* No centralized real-time communication platform exists.

---

## Proposed Solution

The platform provides a centralized system where:

**Donors** can register surplus food
**NGOs** can view and request available food
**Volunteers** manage pickup and delivery
**Admins** monitor redistribution activities

All interactions happen through a simple and efficient web interface.

---

##  Key Features

* ✅ Food donation registration system
* ✅ NGO dashboard for food requests
* ✅ Volunteer assignment management
* ✅ Real-time status updates
* ✅ User-friendly interface
* ✅ Lightweight and fast Flask backend
* ✅ Database-driven workflow

##  Installation & Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/Smart-Food-Redistribution-Platform.git
```

### 2. Navigate to Project Folder

```
cd Smart-Food-Redistribution-Platform
```

### 3️. Install Dependencies

```
pip install flask
```

### 4️. Run the Application

```
python app.py
```

### 5️. Open in Browser

```
http://127.0.0.1:5000
```
